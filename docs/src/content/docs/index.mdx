An **agent** is an LLM running in a loop with tools it can use. The loop continues until the model produces a response without tool calls.

**ia-agents** provides the orchestration layer on top of the [Gemini Interactions API](https://ai.google.dev/gemini-api/docs/interactions) â€” handling the full loop: model calls, tool execution, result submission, and streaming events.

## Packages

| Package | What it provides | Use when |
|---------|------------------|----------|
| [`@philschmid/agents-core`](/packages/agents-core/) | Raw agent loop, tool execution | You want minimal overhead and full control |
| [`@philschmid/agent`](/packages/agent/) | Hooks, built-in tools, skills, subagents | You want lifecycle control and batteries-included |
